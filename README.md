# Basic Environment JavaScript
Babel, BrowserSync, Gulp, Jest, Lint and Watch

### NPM Scripts  
  * `build`
  * `lint`
  * `server`
    * Watch
    * Run server http://localhost:3001
  * `test`
  * `test:coverage`
  * `test:watch`
  * `watch`

### Usage
```bash
# Install
npm install

# Run script
npm run server

# Open in browser
# ==> http://localhost:3001
```

### Folders
  * **./dist** Distribution
  * **./src** Sources
  * **./\_\_tests\_\_** Tests
  * **./\_\_mocks\_\_** Mocks
